# Radar Fundiário · Goiânia

Localizador de imóveis de Goiânia por quadra+lote, endereço, inscrição ou clique no mapa, sobre a camada pública de Cadastro Imobiliário (ArcGIS da Prefeitura). App de arquivo único — abra o [radar-goiania.html](radar-goiania.html) no navegador, sem instalação.

- **Documento de projeto / briefing:** [PROJETO-radar.md](PROJETO-radar.md)
- **Roadmap e decisões técnicas validadas:** [ROADMAP-radar.md](ROADMAP-radar.md)

Dados de imóvel são públicos; o app não coleta nome de titular (consulta manual na fonte oficial). Valores de referência são estimativas, não avaliação oficial.
